## Design Tokens

### Colors (System)
- Use semantic colors: primary, secondary, background
- Selection highlight: accent color with 30% opacity
- Note card background: secondary background color
- Provocation section: slightly darker than note

### Typography
- Document text: System font, user-adjustable size
- Note content: `.body`
- Quoted excerpts: `.callout` with subtle background
- Provocation text: `.body` italic

### Spacing
- Sidebar width: 280pt (collapsible)
- Note card padding: 12pt
- Note card spacing: 8pt
- Minimum touch target: 44pt

### Animations
- Sidebar collapse: 0.2s ease-out
- Scroll to anchor: 0.3s ease-in-out
- Note appear: 0.15s fade
